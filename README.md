# Tic-Tac-Toe Game 🎮

A classic Tic-Tac-Toe game built with HTML, CSS, and JavaScript. This project is a fun, interactive web application that allows two players to compete.

---

## 🚀 Live Demo

**[Click here to play the game!](https://adithya-k05.github.io/Tic-Tac-Toe/)**

---

## ✨ Features

*   **Classic Gameplay**: A standard 3x3 grid for classic Tic-Tac-Toe fun.
*   **Turn Indicator**: Clearly shows which player's turn it is (X or O).
*   **Win Detection**: Automatically detects and announces when a player wins.
*   **Draw Detection**: Declares a draw if the board is filled with no winner.
*   **Reset Button**: A "New Game" button to easily restart the match.
*   **Responsive Design**: Looks great on both desktop and mobile devices.

---

## 🕹️ How to Play

1.  Open the **[Live Demo](https://adithya-k05.github.io/Tic-Tac-Toe/)**.
2.  Player 1 starts as 'X'. Click on any empty square to make a move.
3.  Player 2 is 'O'. Players take turns placing their marks.
4.  The first player to get 3 of their marks in a row (horizontally, vertically, or diagonally) wins the game.
5.  If all 9 squares are filled and no one has won, the game is a draw.
6.  Click the **"New Game"** button at any time to start a new match!
